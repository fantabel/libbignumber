Detailed Description

template<class Blk>
class NumberlikeArray< Blk >

A NumberlikeArray<Blk> object holds a heap-allocated array of Blk with a length and a capacity and provides basic memory management features.

BigUnsigned and BigUnsignedInABase both subclass it.

NumberlikeArray provides no information hiding. Subclasses should use nonpublic inheritance and manually expose members as desired using declarations like this:

public:
NumberlikeArray< the-type-argument >::getLength;
