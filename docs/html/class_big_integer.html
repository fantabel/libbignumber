Detailed Description

A BigInteger object represents a signed integer of size limited only by available memory. BigUnsigneds support most mathematical operators and can be converted to and from most primitive integer types.

A BigInteger is just an aggregate of a BigUnsigned and a sign. (It is no longer derived from BigUnsigned because that led to harmful implicit conversions.)
